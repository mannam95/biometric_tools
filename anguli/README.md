# Anguli Build

## Build the docker image
`docker compose build`

## Spin the container
`docker compose up -d`

## enter into the running container
`docker exec -it anguli-con bash`


##Anguli help

```
Anguli -h // gives you the information available

Anguli -num 2 -npd 8 -ni 8 -outdir /anguli-data

```
